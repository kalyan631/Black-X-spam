BLACKxGOD Telegram Bot

A powerful multi-bot Telegram management system built with Python.

Features

- Multi Bot Support
- Sudo & Owner System
- Spam & Target Commands
- Swipe & React System
- NC & GNC Generator
- Global Commands
- Auto Reply & Utilities
- Easy Render Deployment

Requirements

- Python 3.11+
- Telegram Bot Token

Installation

git clone https://github.com/your-username/your-repo.git
cd your-repo
pip install -r requirements.txt

Environment Variables

Create the following environment variable on Render:

BOT_TOKEN=YOUR_BOT_TOKEN

For multiple bots:

BOT_TOKEN_1=TOKEN1
BOT_TOKEN_2=TOKEN2
BOT_TOKEN_3=TOKEN3

Run Locally

python BLACK_FIXED_OWNER_6982820486.py

or

python main.py

Render Deployment

Build Command

pip install -r requirements.txt

Start Command

python BLACK_FIXED_OWNER_6982820486.py
